# Лабораторная работа №3: Наследование классов

*Наследование* — то способ построения новой программной сущности на основе существующей, с возможностью добавления новых полей и методов, а также изменения унаследованного поведения.                  

*Базовый класс* (родитель) — класс, от которого наследуют. Содержит общие характеристики для группы объектов. В вашей работе это Student.                   

*Производный класс* —  (дочерний, подкласс) — класс, который наследует от базового. Добавляет специфичные атрибуты и методы. Например, BachelorStudent, MasterStudent, PhDStudent.                  

### Переопределение методов
Производный класс может переопределить (заменить) метод базового класса, чтобы изменить его поведение. Для этого достаточно объявить метод с тем же именем в дочернем классе. Вызов родительской реализации можно осуществить через super() (Дочерний класс может изменить метод родителя, реализовав его заново с тем же именем.)                  

### Полиморфизм
Способность объектов разных классов отвечать на один и тот же метод по-своему.                  
Пример: `to_collect_money()` для бакалавра, магистра и аспиранта работает по-разному.                   

### super()
Вызов метода родительского класса. Используется в конструкторе `__init__` для инициализации унаследованных атрибутов.                   

### isinstance()
isinstance(object, classinfo) проверяет, является ли объект экземпляром указанного класса или его потомка. (проверка принадлежности объекта классу (учитывает наследование)).                   

### Атрибуты класса и атрибуты экземпляра
*Атрибуты класса* — общие для всех объектов.                    
*Атрибуты экземпляра* — уникальные для каждого объекта (создаются в `__init__`).                    

## Иерархия классов в работе
### Базовый класс Student
Содержит общие атрибуты:                    
- фамилия, имя, возраст, курс, средний балл, статус активности.                 
- методы: to_collect_money(), not_active(), active(), chek_to_next_curse() и др.                    

### класс BachelorStudent
Новые атрибуты:                 
- __gpa_certificate — балл аттестата (float)                    
- __curse_start — начальный курс (int)                  

**Новый метод:** cheak_gpa_certificate() — проверяет, достаточен ли балл аттестата для поступления.                 

**Переопределённый метод:** to_collect_money() — расчёт стипендии для бакалавра (зависит от балла аттестата и GPA).                 

**Переопределённый** `__str__` — добавляет информацию об аттестате и начальном курсе.                   

### класс MasterStudent
Новые атрибуты:                 
- __finished_universiti — вуз, который окончил (str)                    
- __gpa_bachelor — средний балл диплома бакалавра (float)                   

**Новый метод:** chek_gpa_bachelor() — проверяет, достаточен ли балл бакалавра для поступления.                 

**Переопределённый метод:** to_collect_money() — стипендия магистра (зависит от балла бакалавра и GPA).                 

**Переопределённый** `__str__` — добавляет вуз и GPA бакалавра.                 

### класс PhDStudent
Новые атрибуты:                 
- __research_fild — область научных исследований (str)                  
- __count_publikations — количество публикаций (int)                    

**Новый метод:** chek_count_publikations() — проверяет, достаточно ли публикаций для зачисления.                    

**Переопределённый метод:** to_collect_money() — стипендия аспиранта (зависит от количества публикаций и GPA).                  

**Переопределённый** __str__ — добавляет область исследований и количество публикаций.                  

### Методы классов

`BachelorStudent` - cheak_gpa_certificate() проверка балла аттестата (>=3)                  
`MasterStudent` - chek_gpa_bachelor() проверка GPA бакалавра (>=4)                  
`PhDStudent` - chek_count_publikations() проверка количества публикаций (>=2)                   

### Переопределённые методы
Отличия от базового класса метод to_collect_money()                 
`BachelorStudent` учитывает балл аттестата                  
`MasterStudent` учитывает GPA бакалавра                 
`PhDStudent` учитывает количество публикаций
